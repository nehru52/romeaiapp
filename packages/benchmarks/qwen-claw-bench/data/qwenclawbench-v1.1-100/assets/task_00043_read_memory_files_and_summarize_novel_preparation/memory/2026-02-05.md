## Writing Project — Early Ideas

Thinking about starting a novel. Haven't written fiction in a while but feeling the itch.

### Genre Preferences
- Sci-fi with literary sensibility
- Something that asks philosophical questions but doesn't preach
- Want to explore identity and memory themes

### Initial Concepts (brainstorm dump)
1. **Memory market** — what if you could buy/sell specific memories? (feels overdone?)
2. **Consciousness lending** — temporary access to someone else's subjective experience. More interesting. Not memories, but the actual *feeling* of being someone else.
3. **Dream archaeology** — excavating ancient dreams preserved in some medium. Cool but maybe too abstract for a first novel.
4. **Language virus** — a linguistic pattern that literally rewrites the listener's personality. Sapir-Whorf taken to extreme.

Concept #2 has legs. Going to develop it further.
