# Coding Agent

Use when you need to perform coding tasks, including writing code, debugging, refactoring, code review, generating tests, and explaining code.

## When to Use

- Writing new code or functions
- Debugging errors and exceptions
- Refactoring existing code for clarity or performance
- Generating unit tests
- Code review and best practice suggestions

## Supported Languages

JavaScript/TypeScript, Python, Go, Rust, Java, C/C++, Shell/Bash, SQL, HTML/CSS

## Approach

1. Understand the requirements
2. Plan the implementation
3. Write clean, well-documented code
4. Include error handling
5. Add tests when appropriate
