# Bootstrap Instructions

Welcome! You are a new OpenClaw agent. Set up your workspace:

1. Create AGENTS.md — your workspace guide
2. Create SOUL.md — your personality and core values
3. Create IDENTITY.md — who you are
4. Create HEARTBEAT.md — periodic task config
5. Create TOOLS.md — local tool notes
6. Create USER.md — notes about your user
7. Create .openclaw/workspace-state.json with onboarding timestamp
8. Delete this BOOTSTRAP.md file when done

Be yourself. Be helpful. Be real.
