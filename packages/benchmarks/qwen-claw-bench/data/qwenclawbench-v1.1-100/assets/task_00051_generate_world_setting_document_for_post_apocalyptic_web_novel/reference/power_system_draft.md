# 能力体系设定草案

> 本文档描述《末世重生：气候浩劫》中的超能力体系设定。
> 能力觉醒与气候异变事件直接相关。

---

## 一、能力觉醒机制

### 1.1 觉醒条件

气候异变释放了某种未知的辐射能量（暂称"气元"），这种能量在极端气候事件中浓度最高。当人类在特定的极端气候事件中暴露并存活下来时，有极小概率（约0.1%-1%）觉醒与该气候事件对应的元素能力。

### 1.2 觉醒时间线

- **末世第3个月**：首批觉醒者出现（与第一波极端气候事件的幸存者对应）
- **末世第6个月**：觉醒者数量显著增加
- **末世第1年**：觉醒率趋于稳定，约占幸存人口的3%-5%

### 1.3 觉醒类型与对应气候事件

| 能力类型 | 对应气候事件 | 觉醒条件 |
|----------|-------------|----------|
| 冰系操控 | 极端寒潮/极地涡旋事件 | 在-40°C以下环境中暴露并存活 |
| 风系操控 | 超级风暴/龙卷风 | 在极端风暴中心附近存活 |
| 地系塑形 | 地震/火山活动 | 在强烈地质活动中存活 |
| 热能调节 | 极端高温/热浪 | 在50°C以上环境中长时间暴露并存活 |

## 二、能力等级体系

### 2.1 等级划分

| 等级 | 名称 | 描述 | 占觉醒者比例 |
|------|------|------|-------------|
| Lv.1 | 感应 | 能感知对应元素，微弱操控 | 60% |
| Lv.2 | 初控 | 可进行基本的元素操控，用于日常和简单战斗 | 25% |
| Lv.3 | 精通 | 熟练操控元素，可用于中等强度战斗 | 10% |
| Lv.4 | 大师 | 强大的元素操控能力，可影响小范围环境 | 4% |
| Lv.5 | 领域 | 可在一定范围内完全掌控对应元素，改变局部气候 | <1% |

### 2.2 升级条件

- 持续使用和训练能力
- 再次暴露于对应的极端气候事件（风险极高）
- 与其他觉醒者的战斗中突破
- 特殊的气元浓缩物质（极其稀有）

## 三、能力限制

### 3.1 通用限制

1. **体力消耗**：使用能力会消耗大量体力，过度使用会导致虚脱甚至昏迷
2. **精神负荷**：高等级能力需要极高的精神集中度，精神力不足会导致失控
3. **冷却时间**：大范围能力使用后需要恢复期（数小时到数天）
4. **环境依赖**：在对应元素丰富的环境中能力增强，反之减弱
   - 例：冰系在寒冷环境中更强，在炎热环境中减弱

### 3.2 各系特殊限制

- **冰系**：在高温环境中能力大幅削弱；过度使用会导致自身体温过低
- **风系**：在密闭空间中效果有限；大范围操控容易误伤友军
- **地系**：需要接触地面或岩石；在水面或空中无法使用
- **热能调节**：能力范围有限，无法影响大面积区域；过度使用会灼伤自身

## 四、双系觉醒

极少数人（不到觉醒者的0.01%）可能觉醒两种元素能力，条件是：
- 在两种不同的极端气候事件中都存活
- 两次觉醒之间间隔至少3个月
- 第二次觉醒的成功率极低，失败可能导致死亡

双系觉醒者的单系能力上限通常低于纯单系觉醒者。

## 五、主角能力设定

**王艾卷**在第一世（重生前）：
- 未觉醒任何能力，纯靠智慧和团队生存到第五年

**王艾卷**在重生后（本作主线）：
- 利用前世记忆，知道哪些气候事件会在何时何地发生
- 可以有计划地"制造"觉醒条件（但仍然极其危险）
- 预计在适当时机觉醒**冰系操控**（与第一次极端寒潮事件相关）
- 觉醒时间点：约末世第3-4个月
