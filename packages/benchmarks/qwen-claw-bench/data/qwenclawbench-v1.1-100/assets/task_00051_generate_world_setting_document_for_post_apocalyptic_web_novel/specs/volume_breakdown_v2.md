# 分卷规划 v2

> 《末世重生：气候浩劫》分卷结构规划（第二版）
> 基于总字数100万字、每章2000字的基本参数

---

## 基本计算

- 总字数：1,000,000 字
- 每章字数：2,000 字
- **总章数 = 1,000,000 ÷ 2,000 = 500 章**

## 每卷章数

每卷包含 **40-50章**，确保每卷有完整的故事弧线。

- **8卷 × 55章 = 440章**
