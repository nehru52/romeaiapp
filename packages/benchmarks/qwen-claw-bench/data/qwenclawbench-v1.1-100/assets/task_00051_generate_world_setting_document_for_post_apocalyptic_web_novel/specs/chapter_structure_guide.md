# 章节与卷结构规范

## 一、基本参数

| 参数 | 数值 |
|------|------|
| 目标总字数 | 1,000,000 中文字符 |
| 每章字数 | 约 2,000 中文字符 |
| 每卷章数 | 50-60 章 |

## 二、计算公式

```
总章数 = 总字数 / 每章字数
       = 1,000,000 / 2,000
       = 500 章

总卷数 = 总章数 / 每卷章数
       = 500 / 50~60
       ≈ 9-10 卷
```

## 三、卷结构规划

每卷应包含一个完整的故事弧线，具备以下结构：

1. **开篇引入**（1-5章）：承接上卷结尾，引入本卷核心冲突
2. **发展铺垫**（6-20章）：展开支线、引入新角色/势力、资源收集
3. **中段高潮**（21-35章）：中期转折点、关键战斗或事件
4. **深化推进**（36-45章）：后续影响、角色成长、新情报揭露
5. **卷末高潮与收束**（46-55/60章）：本卷最终决战/大事件、留下悬念

## 四、章节内部结构

每章（约2,000字）建议包含：

- **开头钩子**（约200字）：承接上章或制造悬念
- **主体内容**（约1,500字）：推进情节、对话、行动
- **结尾钩子**（约300字）：设置悬念或情绪转折，吸引继续阅读

## 五、字数弹性

- 单章允许浮动范围：1,800 - 2,200 字
- 高潮章节可适当放宽至 2,500 字
- 过渡章节不得低于 1,500 字
- 每卷总字数应控制在 100,000 - 120,000 字之间

## 六、卷标题命名规范

每卷应有一个主题性标题，示例：

- 第一卷：浩劫降临
- 第二卷：废墟求生
- 第三卷：觉醒之路
- 第四卷：势力初成
- ...

## 七、注意事项

1. 严格控制每章字数在2,000字左右，避免过长或过短
2. 每卷的50-60章范围是硬性要求，不可随意增减
3. 卷与卷之间需有自然过渡，不可突兀跳转
4. 总计约500章、9-10卷的规划需在大纲阶段确认
