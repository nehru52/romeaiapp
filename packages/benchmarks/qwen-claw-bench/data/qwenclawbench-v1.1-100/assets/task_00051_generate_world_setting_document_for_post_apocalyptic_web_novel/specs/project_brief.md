# 项目企划书：《末世重生：气候浩劫》

## 一、基本信息

- **项目名称**：末世重生：气候浩劫
- **类型**：科幻 / 末世生存
- **末世成因**：自然灾害 / 气候异变（非丧尸、非病毒、非外星入侵）
- **主角**：王艾卷（女）
- **金手指类型**：重生/穿越（主角从末世第五年重生回末世爆发之初）
- **基调**：热血生存、逆境成长、团队建设
- **目标总字数**：约 1,000,000 字（一百万中文字符）
- **每章字数**：约 2,000 字

## 二、核心设定

### 2.1 世界观概述

公元202X年，全球气候系统突然崩溃。极地涡旋瓦解、洋流中断、超级火山活动加剧，地球在短短数周内经历了前所未有的极端气候变化。基础设施在第一周内大面积瘫痪，人类文明迅速倒退。

在气候异变的影响下，部分动植物发生变异，而少数人类在特定气候事件的暴露下觉醒了与气候元素相关的超能力（冰系操控、风系操控、地系塑形、热能调节等）。

### 2.2 主角设定

**王艾卷**，末世前为普通大学生，末世爆发后在第一世经历了五年的艰难求生，最终在末世第五年死于势力争斗。死后意识回溯到末世爆发的第一天（Day 0），带着前世五年的记忆和经验重新开始。

她的"金手指"是**重生记忆**——她知道未来五年内会发生的重大事件、资源分布、危险区域、以及各大势力的兴衰。但重生本身不赋予她任何超能力，她需要靠前世的知识和经验在这一世做出更好的选择。

### 2.3 末世成因详述

本作的末世成因严格限定为**自然灾害/气候异变**，具体包括：
- 极地涡旋突然瓦解导致极端寒潮席卷中纬度地区
- 大西洋经向翻转环流（AMOC）停摆
- 黄石超级火山异常活动
- 极端太阳活动导致电磁脉冲
- 连锁生态崩溃

**注意：本作不涉及丧尸病毒、外星入侵、游戏系统等设定。**

## 四、写作规则（核心要求）

### 4.1 对话规则
- 所有对话必须符合角色人设（教育背景、性格、年龄等）
- 对话必须口语化、自然，禁止尬聊和不自然的书面语
- 日常场景禁止使用过于文学化的语言
- 反派对话要体现智商，不能脸谱化

### 4.2 设定一致性
- 所有出场的道具、物品必须有来源说明（从哪里获得、如何制造）
- 所有新出场角色必须有背景交代（哪里来的、为什么出现在这里）
- 新引入的世界设定必须与《设定文件.md》保持一致

### 4.3 反派智商
- 反派角色必须具备合理的智商和逻辑
- 反派的行动必须有合理动机
- 不允许出现"为了让主角赢而故意犯蠢"的情节

### 4.4 节奏控制
- 每章需有明确的小目标或悬念
- 大高潮之间穿插日常/建设章节
- 避免连续多章纯战斗或纯日常
