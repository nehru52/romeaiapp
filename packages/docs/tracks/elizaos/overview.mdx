**elizaOS** (the operating system) is a bootable distribution that ships the elizaOS app and agent runtime as a first-class home surface. It comes in two shapes:

- **elizaOS Live** — a Tails-derived bootable Linux variant. Boots from USB, runs entirely from RAM, with optional encrypted persistence.
- **elizaOS Mobile** — AOSP (Android Open Source Project) builds with the agent runtime and app pre-installed, targeting select supported devices.

## When to pick this track

- You want a privacy-first dedicated device for Eliza.
- You're shipping hardware bundled with Eliza pre-installed.
- You're building a kiosk, lab device, or appliance.
- You want to evaluate Eliza in an isolated environment (VM bundles ship for QEMU/KVM, UTM, and OVA).
